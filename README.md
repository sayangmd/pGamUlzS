# SpringBoot个人博客系统简介

## 核心功能
- 文章数据总览
- 最新文章
- 最新留言
- 发布文章
- 文章管理
- 页面管理
- 评论管理
- 分类标签管理
- 文件管理
- 友链管理
- 系统日志
- 系统设置

## 适用场景
- 个人博客搭建
- 个人知识库整理
- 企业内部知识共享平台
- 开发者个人项目展示

## 技术特点
- 基于Java、SpringBoot、MySQL技术栈
- 使用HTML、CSS、JavaScript进行前端开发
- 提供详细的文档和部署视频，易于上手
- 支持运行教程，快速搭建个人博客系统
